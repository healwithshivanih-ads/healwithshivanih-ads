My recommendation

Beighton: Option A — composite CC0 image for the form. Single image, all 5 tests, clean line-drawing style, no attribution complexity.

NASA lean test: keep my hand-drawn SVG (the one in the previous preview). Wikimedia's tilt-table option would mislead clients.

Fallback if Option A's style feels off to you: Option B's 5 photos work too — each per-question — but mean five separate image attributions, and clients see real photos of strangers' joints (some find this off-putting). Option A keeps it abstract.
