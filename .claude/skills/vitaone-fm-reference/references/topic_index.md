## Symptom-Cluster Index

### Fatigue / low energy
- **Practice Guide sections:** Sleep, Nervous System, Thyroid (if cold + hair loss), Metabolic (if post-meal crash), Nutrients (iron, B12, B1)
- **Evidence tiers:** Thyroid §2, Insulin/Diabetes §12, Inflammation (if chronic)

- **Rule out first:** thyroid (full panel), ferritin, B12, vitamin D, sleep apnea screening, depression

### Weight gain / metabolic
- **Practice Guide:** Metabolic, Perimenopause (if age 35+), PCOS (if women + irregular cycles)

## Red-Flag / Refer-Out Index

When any of these appear in a client description, the response MUST surface the referral trigger prominently — even if other topics are more prominent in the case.

- **Chest pain, severe breathing issues, suspected stroke** → emergency care
- **Suicidality / active self-harm** → emergency/mental health crisis
- **Suspected T1D** (child/young adult, unexplained weight loss, polyuria, polydipsia) → emergency endocrinology
- **Active DKA symptoms** (fruity breath, vomiting, confusion in diabetic) → emergency
- **Sudden severe abdominal pain** → ER
- **Heavy or prolonged uterine bleeding** (>7 days, >1 pad/hour) → gynecology urgent
- **Post-menopausal bleeding** → gynecology urgent
- **New or worsening neurological symptoms** → neurology
- **Unexplained weight loss, night sweats, persistent lumps** → medical workup for malignancy
- **Suspected eating disorder** → mental health specialist
- **Pregnancy with any metabolic/hormonal concern** → OB-GYN first
- **Blood in stool, persistent severe GI pain** → GI
- **Active autoimmune flare** → rheumatologist / specialist
- **Cardiac family history + new lipid concerns** → cardiology
- **Client asks "should I stop my medication?"** → prescribing clinician only

## Quick "what to read first" decision tree

1. **Is this a client case?** → Read `practice_guide.md` relevant topic section, then check `evidence_tiers.md` for any specific claims you'll reference.
2. **Is this an evidence-check on a specific claim?** → `evidence_tiers.md` directly, matching the topic.
3. **Is this a "what does FM say about X" question?** → `practice_guide.md` if X is in the 7 topics; otherwise `full_kb.md`.
4. **Is this about supplement forms, specific nutrients, lab panels, or less-common topics?** → `full_kb.md` (use its section index).
5. **Is this a scope question ("can I recommend X")?** → `practice_guide.md` Parts A-D.
